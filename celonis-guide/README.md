## Cómo Usar Esta Guía

1. **Lectura secuencial:** Sigue los capítulos en orden para un recorrido completo del programa Celonis Kickstarter.
2. **Referencia rápida:** Usa el índice para saltar al tema que necesites consultar.
3. **Navegación:** Cada capítulo incluye enlaces al anterior, al siguiente y de vuelta al índice.
